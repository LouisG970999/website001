Local data controls

The app includes local backup export/import, scan history controls, known-parts library controls, test log export, and full local data deletion from Settings.

AI result disclaimer

TechSpec Scanner uses AI to assist with mechanical component identification. Results can be incomplete or incorrect. Verify with physical measurements, markings, manufacturer documentation, and engineering judgment before using results for safety-critical, purchasing, repair, or design decisions.

Current capabilities

Scanning Guided photo slots for overview, markings, side view, and scale reference.
AI analysis Mechanical component identification using a backend-protected Gemini API key.
Reports Editable results, technician notes, JSON export, CSV test log, and PDF report export.
Cost controls Server usage limits, request size limits, anonymous install ID, and per-device scan counters.
