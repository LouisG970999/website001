Beta Test Guide

Use this checklist to test TechSpec Scanner and send feedback that is useful for improving the first public version.

5-10 minute test Mechanical parts preferred Human verification required

Before you test

TechSpec Scanner is an AI-assisted beta. Results can be incomplete or incorrect. Do not use the app as the sole basis for safety-critical, repair, purchasing, or design decisions.

Do not upload confidential documents, private API keys, passwords, unrelated personal data, or components you are not allowed to share.

1. Pick test parts

  • One easy, recognizable part such as a bolt, bearing, fitting, gear, or tool.
  • One harder part with fewer markings or unclear geometry.
  • One part with visible markings, a label, or a stamped code if possible.

2. Take useful photos

  • Start with one sharp overview photo.
  • Add markings or labels if they exist.
  • Add a side view or scale/ruler when dimensions matter.
  • Use bright light and a plain background.

3. Judge the result

  • Is the component name plausible?
  • Are function, material, and visible features useful?
  • Does the confidence feel honest?
  • Does the app ask for the right next photo?

What feedback helps most?

Wrong result Tell us what the part actually was and which clue the app missed.
Unclear result Tell us which words, cards, buttons, or warnings felt confusing.
Good result Tell us why it was useful, so the app can preserve that behavior.
Bug Tell us what device/browser you used and what you clicked before the issue happened.

Ready?

Open the beta app, run a few scans, then send feedback. If you opened the feedback form from a scan result, the scan context will already be filled in.
