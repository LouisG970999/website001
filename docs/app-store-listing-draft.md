## Description

TechSpec Scanner helps technicians, students, makers, and engineers document and identify mechanical components from workshop photos.

Take or upload component images, add context notes and measurements, and receive a structured AI-assisted result with likely component type, visible features, material clues, possible standards or part numbers, uncertainty warnings, and recommended checks.

Built for practical technical work, TechSpec Scanner supports guided multi-photo scans for overview, markings, side view, and scale references. Results can be edited, evaluated, saved as known parts, matched against a local library, and exported as reports.

Key features:

- Guided component scanning from iPhone camera photos
- AI-assisted mechanical part identification
- Specialist scan modes for fasteners, bearings, gears, shafts, fluid power parts, sensors, seals, springs, brackets, conveyor parts, and linear guides
- Manual measurements and context notes
- Editable scan results with verification status
- Known-parts library and local matching
- Reports with JSON, CSV test log, and PDF export
- Evaluation workflow to mark results as correct, partly correct, or wrong
- Usage controls designed to protect API costs

Important: TechSpec Scanner provides AI-assisted technical suggestions, not certified engineering verification. Always confirm results using physical measurements, markings, manufacturer documentation, and professional judgment before using them for safety-critical, purchasing, repair, or design decisions.
